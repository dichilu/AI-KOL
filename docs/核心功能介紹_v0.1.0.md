# 核心功能介紹｜v0.1.0

**版本**：v0.1.0
**日期**：2026-05-05
**狀態**：專案初始化，架構確認

---

## 專案目的

自動找到台灣工業控制領域中，真正受工程師信任的意見領袖（KOL），協助行銷人員進行有效率的 B2B 聯繫與合作。

---

## 資料來源

| 平台 | 收集方式 |
|---|---|
| YouTube | YouTube Data API v3 |
| Facebook | Playwright 瀏覽器自動化（第二階段）|

---

## CrewAI 評估架構

| Agent | 角色 | 任務 |
|---|---|---|
| Agent 1 | 內容與專業度分析師 | 判斷內容是否真的在講工控、是真工程師還是表面科普 |
| Agent 2 | 數據分析師 | 評估互動率趨勢、識別假帳號或殭屍粉 |
| Agent 3 | 推薦決策師（老闆）| 綜合兩份報告 + 歷史合作經驗，給出 A/B/C 等級與推薦理由 |

**使用模型**：claude-sonnet-4-6
