# 核心功能介紹｜v0.2.0

**版本**：v0.2.0
**日期**：2026-05-05
**狀態**：本機資料收集與 CrewAI 評估流程完成

---

## 新增功能（相較 v0.1.0）

- `youtube_collector.py`：從 Mock 資料收集 YouTube KOL 欄位，含 Email/電話正規表達式擷取
- `crewai_evaluator.py`：三 Agent 評估流程（Mock 模式），輸出 A/B/C 等級與推薦理由
- Mock 資料集：6 筆模擬 YouTube 頻道（含工控相關與無關頻道，用於驗證篩選邏輯）

## 執行方式

```bash
cd scripts
python -X utf8 crewai_evaluator.py
```

## 資料流

```
youtube_mock.json
    ↓ youtube_collector.py（收集、計算、擷取聯絡資訊）
    ↓ crewai_evaluator.py（三 Agent 評估）
    → 輸出：A/B/C 等級 + 推薦理由
```

## 評估 Agent 架構

| Agent | 角色 | 輸入 | 輸出 |
|---|---|---|---|
| Agent 1 | 內容與專業度分析師 | 頻道名稱、簡介、標籤 | 內容分析報告 |
| Agent 2 | 數據分析師 | 訂閱數、觀看數、互動率 | 數據健康報告 |
| Agent 3 | 推薦決策師（老闆）| 兩份報告 + 歷史合作回饋 | A/B/C + 推薦理由 |

## 切換真實 API

將兩個腳本頂部的 `MOCK_MODE = True` 改為 `False`，並設定環境變數：
```
ANTHROPIC_API_KEY=your_key
```
